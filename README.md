# ParserXML

Задача:
Консольная программа, подсчитывающая количество уникальных значений атрибута name тега modification.

Пример запуска:

java -jar myprog.jar cars.xml



Примечания:
 * Подсчет ведется без учета принадлежности маркам и моделям, т.е., например, написание "1.6 MT (80 л.с.)" подсчитывается 1 раз, сколько бы оно не встречалось.
